# TV Explorer

## Overview

Next.js TV discovery app for exploring shows and viewing details through a simple frontend interface.

Project type: JavaScript/React application.

- `src/components/DetailsDrawer.jsx`

- `src/styles/global.css`

## Tech Stack

Next.js, React

## How To Run / Use

```bash
npm install
npm run dev
```

For production validation, run:

```bash
npm run build
```
